# QR Code Generator

This is a simple QR Code Generator built using plain HTML, CSS and JavaScript.  
You can type any text or URL, click **Generate**, and it will create a QR code for you on the page.

## Features

- Enter any text or URL
- Generates a QR code instantly
- Clean and minimal UI
- Uses the `qrcodejs` library (loaded from CDN)

## Tech Stack

- **HTML** – structure
- **CSS** – basic styling
- **JavaScript** – QR code generation logic
